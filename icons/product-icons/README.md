# Product icon SVGs

Drop product (UI) icon SVG sources here, **one file per codicon ID** (e.g. `chevron-down.svg`, `git-branch.svg`, `gear.svg`).

- 16×16 viewBox, single color (`currentColor` / black fill).
- Filename must equal the codicon ID — the build pipeline reads the filename to assign the correct PUA codepoint when generating the font.

See [../../docs/product-icons/product-icons-list.md](../../docs/product-icons/product-icons-list.md) for the full target list and [../../docs/product-icons/how-product-icons-work.md](../../docs/product-icons/how-product-icons-work.md) for the font/theme pipeline.
