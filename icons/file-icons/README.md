# File icon SVGs

Drop file/folder icon SVG sources here. Filenames should match the keys used in `themes/makinda-file-icon-theme.json` (e.g. `javascript.svg`, `folder-src.svg`, `folder-src-open.svg`).

See [../../docs/file-icons/file-icons-list.md](../../docs/file-icons/file-icons-list.md) for the full target list and [../../docs/file-icons/how-file-icons-work.md](../../docs/file-icons/how-file-icons-work.md) for sizing/authoring rules.
